# Dementia-using-machine-learning
Detection of Dementia using machine learning and python 

-This project focuses on detecting dementia using machine learning techniques based on MRI and clinical data.

-The model is trained using the OASIS dataset (235 instances) containing demographic, clinical, and morphometric features.

-Machine learning models like Multilayer Perceptron (ANN) and J48 Decision Tree are used for classification.

-Initial experiments showed a 48% error rate, which was reduced to 17% error after applying image processing and Voxel-Based Morphometry (VBM) techniques.

-The final model achieved around 83% accuracy (reliability) in dementia prediction.

-The system uses features like age, cognitive scores, and brain morphometric data (WM, GM, CSF) for classification.

-The objective is to support early detection of dementia and improve diagnosis efficiency using AI.

-This project demonstrates the application of machine learning in healthcare for better decision support systems.
